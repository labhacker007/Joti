## RESPONSIVE BREAKPOINTS

### Tailwind Breakpoints

```
xs:  320px   (extra small phones)
sm:  640px   (small phones)
md:  768px   (tablets)
lg:  1024px  (small laptops)
xl:  1280px  (laptops)
2xl: 1536px  (desktops)
```

### Login Page Responsiveness

```
xs/sm (< 640px):
- Single column layout
- No left brand panel (mobile-only form)
- Full-width inputs
- Theme grid: 3x2 (6 themes)

md (768px - 1023px):
- Two columns starting to show
- Left panel hidden, form centered
- Width: ~500px max

lg (1024px+):
- Full two-column layout
- Left 50%: Brand messaging
- Right 50%: Login form
- Sidebar becomes visible
```

### Navbar Responsiveness

```
xs/sm (< 640px):
- Hamburger menu for navigation
- Theme icon still visible
- Condensed layout
- Single row

md (768px+):
- Full horizontal navigation
- All items visible
- Theme switcher in top-right
- Two-row layout (logo + nav, actions)

lg (1024px+):
- Full layout with all spacing
- Centered navigation
- Icon theme switcher + user menu
- Single row
```
